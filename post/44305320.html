数据类型

Python是一种动态类型语言，这意味着在编写代码时不需要显式地声明变量的数据类型。我们可以通过type函数来查看变量的数据类型。

直接在type的括号里加上变量即可

加前缀0o是八进制

加前缀0b是二进制

运行时自动转换为十进制输出

关于整形内存的问题

一个整形最少占用24字节（数字0），从1开始占用28字节，到了2

的30次方占用32字节，而2的128次方占用144字节。得到两个规律

  1. 字节随着数字增大自动增加。
  2. 每次增量是4个字节

字符串

python中的单引号和双引号包裹的都是字符串，python中不支持单字符类型

当字符串里有单引号时可以用双引号包裹，或者转义单引号

当字符串里有双引号时可以用单引号包裹

当字符串内容比较复杂时（比如既有单引号和双引号），可以用三个单引号或者双引号包裹起来

在字符串前面加’r‘可以使整个字符串不被转义

字符串驻留机制

Python仅保存一份相同且不可变字符串，不用的值被存放在字符串的驻留池中，Python的驻留机制对相同字符串只保留一份拷贝，后续创建相同的字符串时，不会开辟新控制，而是把该字符串的地址赋给新闯将的变量

驻留机制的几种情况：

  1. 字符串是由26个英文字母大小写，0-9,_组成
  2. 字符串长度为0或者1
  3. 字符串在编译时进行驻留，而给运行时
  4. [-5,256]的整型数字

比较运算符

与其他语言不同是

python中有is和is not两个新的运算符

is =>判断两个变量对象是否为用一个

is not=>判断两个变量引用对象是否不同

即判断变量的指向的地址是否相同

c ** = a 等效于 c = c ** a

三元运算符

python没有引入 ? : 这个运算符，而是通过if else关键词来实现相同的功能

max = a if a>b else b

即如果a > b 那么把a的值赋给max，否则把b的值赋给max
